# MEDEVAK — Стиль (референс: vovkydavinci.army)

Стиль сформовано на основі [vovkydavinci.army](https://vovkydavinci.army/) — батальйон «Вовки Да Вінчі».

---

## 1. Кольорова палітра

| Роль | Значення | Використання |
|------|----------|--------------|
| **Background** | `#000000` | Основний фон |
| **Surface** | `#1a1a1a` — `#2a2a2a` | Картки, панелі |
| **Surface light** | `#c0c0c0` — `#e5e5e5` | Картки на темному (вакансії) |
| **Text primary** | `#ffffff` | Заголовки, основний текст |
| **Text secondary** | `#a3a3a3` | Підзаголовки, опис |
| **Accent** | `#ffffff` | CTA, посилання, акценти |
| **Border** | `#404040` | Роздільники, контури |

**Принцип:** високий контраст, чорно-біла основа, мінімум кольору.

---

## 2. Типографіка

| Елемент | Шрифт | Вага | Регістр | Розмір |
|---------|-------|------|---------|--------|
| **H1** | Sans-serif (Montserrat, Inter, system-ui) | 700 | UPPERCASE | 2rem — 2.5rem |
| **H2** | Sans-serif | 600–700 | UPPERCASE | 1.5rem |
| **H3** | Sans-serif | 600 | UPPERCASE | 1.25rem |
| **Body** | Sans-serif | 400 | normal | 1rem |
| **Caption** | Sans-serif | 400 | normal | 0.875rem |
| **Numbers** | Sans-serif | 400 | normal | 1rem |

**Правила:**
- Заголовки — UPPERCASE, жирні
- Текст — нормальний регістр
- Мінімалізм, без декоративних шрифтів

---

## 3. Компоненти

### Картки
- Фон: світло-сірий (`#e5e5e5`) на чорному або темно-сірий (`#1a1a1a`) на чорному
- Геометрія: прямокутник з **скосом** (chamfer) у кутах — невеликий трикутний зріз
- Відступи: `padding: 1.5rem`
- Тінь: мінімальна або відсутня

### CTA (кнопки / посилання)
- Текст: `долучитись →` або `Спробувати себе`
- Стрілка `→` для дії
- Контрастний колір (білий на темному)

### Роздільники
- Тонка вертикальна/горизонтальна лінія `1px solid #404040` або `#ffffff`

---

## 4. Layout

- **Grid:** 2 колонки для списків (вакансії, кейси)
- **Вирівнювання:** контент по центру, асиметрія дозволена
- **Простір:** багато «чорного» простору, без перевантаження
- **Секції:** чіткі блоки з заголовками `| СЕКЦІЯ` або `//01`, `//02`

---

## 5. Тон і атмосфера

- Серйозний, професійний
- Мінімалізм, функціональність
- Військова/медична естетика
- Без зайвих декорацій

---

```css
  --vdv-bg: #000000;

  --vdv-border: #404040;
  --vdv-font: 'Inter', 'Montserrat', system-ui, sans-serif;
}
```

---

## 7. Зображення та фото

- **Великі фото:** не використовувати
- **Іконки:** мінімальні, монохромні (білі на темному)
- **Аватари / превʼю:** невеликі, до 48–64px
- **Фон:** суцільний колір, без фото-фонів

---

## 8. Референси

- [vovkydavinci.army](https://vovkydavinci.army/) — головна
- Вакансії — сітчастий список з картками
- Медична служба «Ульф» — великі заголовки, фото, асиметричний layout
- Патронатна служба — емблема, мінімалізм, завантаження
- **Без великих фото** — тільки іконки та невеликі превʼю
